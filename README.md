# tears-of-the-kingdom-language
A tool for writing and editing in the Zonai language of the upcoming game The Legend of Zelda Tears of the Kingdom.

To add a letter simply click on one of the symbols in the bottom of the window.
To navigate use the arrow keys.
To add a new page use the "insert" key.
To delete a page use the "delete" key.
Enter creates a new column.
Backspace deletes both letters and columns when no letters are present in them.
Control + s saves your work while loading happens automatically at the start of the program.
The savefile "text.txt" needs to be in the same folder as the .exe itself.
